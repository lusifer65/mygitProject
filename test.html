configure the name and email on git on local machine
  • for name:- git config --global user.name "your name"
  • for email:-git config --global user.email "your email"
  • for show the configuration:- git config --global user.<name,email>
flow in github 
            1-working to local repo:-
                working directory ---(git add)---->stagingArea----(git commit)---->local repostary
            2.local repo to remote (Github)
                local repo -------(git push)---->remote server 
                remote server------(git pull)---->local server
            3.for knowning status :- 
                git status
how to add file on git 
            1-create a folder 
            2-open cmd 
            3-run command-> git init
            4-make some change in file inside the folder
            5-run command-> git add .
            6-run command-> git commit -m "message"
how to add remote location(github) 
            after commit in git:-(above step)
                1-open git and create new repo 
                2-copy the link by selecting https option
                3-run command-> git remote "LINK"
